WandB Board

Starts the local server from the current directory

wandb board

Starts the server on top of a custom directory

wandb board --logdir=~/other/directory

WandB board is an open source and local version of our metric tracking solution. It scans a local directory for existing and new runs and starts a webserver that provides the interface and backend.
